About Me

Hi! I'm Tiffani Hess. A web developer with a passion for all things front and back end.

I'm currently a freelance web developer living in Waterloo, Iowa. I enjoy programming for both the client and server side. When I'm not building applications, my free time is spent learning new technologies and superior ways of implementing the ones I'm already putting into practice.


My most recent projects utilize React, Styled Components, Node, Express, PostgreSQL, JSX, HTML, CSS, and Axios. I also have experience in a number of other technologies such as MongoDB, EJS, Heroku, Git, NPM, Amazon S3, Google APIs, Sentry, SendGrid, Mailgun, Bootstrap, Material UI, and Semantic UI.
